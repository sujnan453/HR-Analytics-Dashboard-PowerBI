# HR Analytics Dashboard – Power BI

An interactive HR Analytics Dashboard built using **Power BI** and IBM HR Employee data, designed to provide actionable insights into workforce trends, attrition, and compensation.

---

## 📁 Project Structure

```
HR-Analytics-Dashboard-PowerBI/
│
├── HR Analytics Dashboard.pbix     # Power BI dashboard file
├── IBM_HR_Dataset.xlsx              # Raw IBM HR employee dataset
├── Dashboard_Screenshot.png         # Dashboard preview image
└── README.md                        # Project documentation
```

---

## 📊 Dataset Overview

- **Source:** IBM HR Employee Attrition Dataset
- **Records:** 23,222 employees
- **Columns:** 41 features including Age, Department, Job Role, Monthly Income, Attrition, and more

---

## 📌 Dashboard Features

- **Attrition Analysis** – Tracks voluntary resignation rate by department, gender, and job role
- **Department Breakdown** – R&D (65.2%), Sales (30.4%), HR (4.4%)
- **Income Analysis** – Average monthly income by department and job level
- **Hiring Cost by Platform** – Compares recruitment cost across platforms (LinkedIn, Indeed, Referral, etc.)
- **Workforce Demographics** – Age, gender, marital status, education field distribution
- **KPI Cards** – Revenue, attrition rate, revenue per employee, total compensation cost

## 🛠️ Tools Used

- **Power BI Desktop** – Dashboard creation and visualization

## 🚀 How to Use

1. Clone or download this repository
2. Open `HR Analytics Dashboard.pbix` in **Power BI Desktop**
3. The dataset is pre-loaded — explore the interactive visuals
4. Use slicers to filter by Department, Gender, Job Role, and more
